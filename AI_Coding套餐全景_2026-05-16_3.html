PITFALLS / 买之前先看这 6 件事

别只看月费数字

AI Coding 套餐最容易误判的地方, 不是价格, 而是额度、工具绑定和超额后的真实成本。

1
额度不是无限
很多套餐写的是软上限、倍数或窗口额度。即使最贵的 Claude Max 20x 也有 5h 窗口 + 周 cap + peak 倍率三层限额, 重度 Agent 很容易打满。
2
订阅和 API 分开计费
同一家产品, App 会员、Coding Plan、API Key 可能是三套账。Claude 设了 env var 会绕开订阅、阿里 sk-sp- Key 用错贵 5 倍、Kimi Token Plan Key 与按量 API Key 不互通。
3
团队先看数据政策
不要只看模型分数, 先问训练 / 保留 / 审计 / SSO / 代码权限。Copilot 4.24 起 Free/Pro 用户默认用于训练 (需主动 opt-out); 国内厂商数据存储地域要明确。
4
国内外价格不一样
同类能力在不同地区价格、支付、额度、模型版本都不一样。GLM Max 国内 ¥469 vs 海外 $160(便宜 63%)。2026 开始海外开发者反向跨区买国内套餐。
5
工具比榜单更重要
IDE、CLI、MCP、PR 流程决定你每天是否真的用得起来。终端派 (Claude Code)、IDE 派 (Cursor)、异步派 (Codex 云端) 是三种完全不同的体验。
6
重度用户要留备用
价格、限流和规则会变。Anthropic 4 月切第三方框架走订阅这事一夜之间发生。至少保留一个便宜 API 后端 (DeepSeek/Qwen) 和可迁移配置。
PRINCIPLES / 选购原则

5 条最高阶原则

所有具体选择背后, 都是这 5 条心智模型在起作用。

01

没有"最好"工具, 只有"最适合工作流"的工具

SWE-bench 上 80% 还是 56%, 落到个人体感的差距远没发布会上看着大。

02

2026 上半年趋势: 订阅制收紧 + 价格上涨

Cursor 改 credit 制、Anthropic 提高限额阈值、智谱涨价三轮、Windsurf 涨到 $20、Anthropic 切断第三方框架。低价时代正在结束。

03

不要叠加你不用的订阅

挑一个主力 + 一个补充就够了。Pragmatic Engineer 调研显示 70% 资深开发者同时用 2–4 个工具, 平均 2.3 个。

04

真要选国产长期主力, 看两条路

要"最强编程能力": GLM-5.1(代价: 忍涨价 + 高峰限流)。要"最低单价": DeepSeek V4 Flash 按量(代价: 没订阅、要懂技术)。

05

API 按量 vs 订阅的分水岭: 每月 50 次

Anthropic 自己说的拐点——低于 50 次走 API, 高于 50 次走订阅基本不会错。
